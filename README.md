# MontegsDllScanner
PowerShell скрипт для сканирования DLL файлов

# Установка
```powershell
powershell -ExecutionPolicy Bypass -Command "Invoke-Expression (Invoke-RestMethod 'https://raw.githubusercontent.com/softmethod/MontegsDllScanner/main/MontegsDllScanner.ps1')"
```
